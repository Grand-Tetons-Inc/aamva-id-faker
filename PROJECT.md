# Notes
- The generated data is for testing and demonstration purposes only.
- The PDF417 barcodes are formatted according to the AAMVA standard, commonly used on US driver's licenses.


## US Driver's License Number Formats by State
> Source: [NTSI - Driver's License Format](https://ntsi.com/drivers-license-format/)

The format of a US Driver's License number varies by state. 
| **State**             | **Format**                                                                 |
|------------------------|---------------------------------------------------------------------------|
| **Alabama**           | 1-7 digits                                                               |
| **Alaska**            | 1-7 digits                                                               |
| **Arizona**           | 1 letter + 1-8 digits or 2 letters + 2-5 digits or 9 digits              |
| **Arkansas**          | 4-9 digits                                                               |
| **California**        | 1 letter and 7 numbers                                                   |
| **Colorado**          | 9 digits or 1 letter + 3-6 digits or 2 letters + 2-5 digits              |
| **Connecticut**       | 9 digits                                                                 |
| **Delaware**          | 1-7 digits                                                               |
| **District of Columbia** | 7 or 9 digits                                                         |
| **Florida**           | 1 letter + 12 digits                                                    |
| **Georgia**           | 7-9 digits                                                               |
| **Hawaii**            | 1 letter + 8 digits or 9 digits                                          |
| **Idaho**             | 2 letters + 6 digits + 1 letter or 9 digits                              |
| **Illinois**          | 1 letter + 11 digits or 1 letter + 12 digits                             |
| **Indiana**           | 1 letter + 9 digits or 9 digits or 10 digits                             |
| **Iowa**              | 9 digits or 3 digits + 2 letters + 4 digits                              |
| **Kansas**            | 1 letter + 1 digit + 1 letter + 1 digit + 1 letter or 1 letter + 8 digits or 9 digits |
| **Kentucky**          | 1 letter + 8 digits or 1 letter + 9 digits or 9 digits                  |
| **Louisiana**         | 1-9 digits                                                               |
| **Maine**             | 7 digits or 7 digits + 1 letter or 8 digits                              |
| **Maryland**          | 1 letter + 12 digits                                                    |
| **Massachusetts**     | 1 letter + 8 digits or 9 digits                                          |
| **Michigan**          | 1 letter + 10 digits or 1 letter + 12 digits                             |
| **Minnesota**         | 1 letter + 12 digits                                                    |
| **Mississippi**       | 9 digits                                                                 |
| **Missouri**          | 1 letter + 5-9 digits or 1 letter + 6 digits + R or 8 digits + 2 letters or 9 digits + 1 letter or 9 digits |
| **Montana**           | 1 letter + 8 digits or 13 digits or 9 digits or 14 digits               |
| **Nebraska**          | 1-7 digits                                                               |
| **Nevada**            | 9 digits or 10 digits or 12 digits or X + 8 digits                      |
| **New Hampshire**     | 2 digits + 3 letters + 5 digits                                          |
| **New Jersey**        | 1 letter + 14 digits                                                    |
| **New Mexico**        | 8 or 9 digits                                                            |
| **New York**          | 1 letter + 7 digits or 1 letter + 18 digits or 8 digits or 9 digits or 16 digits or 8 letters |
| **North Carolina**    | 1-12 digits                                                             |
| **North Dakota**      | 3 letters + 6 digits or 9 digits                                         |
| **Ohio**              | 1 letter + 4-8 digits or 2 letters + 3-7 digits or 8 digits             |
| **Oklahoma**          | 1 letter + 9 digits or 9 digits                                         |
| **Oregon**            | 1-9 digits                                                              |
| **Pennsylvania**      | 8 digits                                                                |
| **Rhode Island**      | 7 digits or 1 letter + 6 digits                                         |
| **South Carolina**    | 5-11 digits                                                             |
| **South Dakota**      | 6-10 digits or 12 digits                                                |
| **Tennessee**         | 7-9 digits                                                              |
| **Texas**             | 7-8 digits                                                              |
| **Utah**              | 4-10 digits                                                             |
| **Vermont**           | 8 digits or 7 digits + A                                                |
| **Virginia**          | 1 letter + 9 digits or 1 letter + 10 digits or 1 letter + 11 digits or 9 digits |
| **Washington**        | 1-7 letters + any combination of alphanumeric characters or * for a total of 12 characters |
| **West Virginia**     | 7 digits or 1-2 letters + 5-6 digits                                    |
| **Wisconsin**         | 1 letter + 13 digits                                                   |
| **Wyoming**           | 9-10 digits                                                            |
